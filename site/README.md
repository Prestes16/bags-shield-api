# Bags Shield Site

Documentation and landing page for Bags Shield, built with Docusaurus.

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Deploy

This site is configured for Cloudflare Pages:

1. Connect your GitHub repository to Cloudflare Pages
2. Set build command: `npm run build`
3. Set output directory: `build`
4. Add custom domains in Cloudflare Pages dashboard

## Structure

- `docs/` - Documentation pages
- `src/pages/` - React pages (landing, etc.)
- `static/` - Static assets (images, etc.)
- `docusaurus.config.js` - Docusaurus configuration

## Theme

Navy + Glow Cyan + Glass effect:

- Background: `#020617` / `#07122a`
- Accents: `#2FA7FF` / `#4DD4FF`
- Glass cards with blur and subtle borders
